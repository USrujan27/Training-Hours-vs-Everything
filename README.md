# Training Hours vs Everything

## Overview
This project analyzes the impact of AI training hours on business performance using data analytics, machine learning, and interactive visualizations.

## Features
- Industry-wise Analysis
- Country-wise Analysis
- Correlation Score Calculation
- Interactive Dashboard
- Business Insights
- Comparative Trend Analysis

## Technology Stack
- React.js
- Recharts
- FastAPI
- Pandas
- XGBoost
- Random Forest
- Gradient Boosting
- Monte Carlo Simulation
- SHAP Explainability
